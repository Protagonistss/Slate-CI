# Slate

Slate 是一个由 AI 驱动的现代化代码编辑器，通过深度集成的 AI Agent 与工具链，为开发者提供更智能、更高效的编程体验。

## 核心特性

- **AI Agent 执行**：内置自主执行 Agent，可根据自然语言描述完成复杂工程任务
- **Monaco 编辑器**：与 VS Code 相同的编辑器内核，支持丰富的语言特性
- **MCP 支持**：基于 Model Context Protocol，可扩展 AI 的工具能力
- **跨平台**：基于 Tauri，支持 Windows 与 macOS

## 下载

安装包由本仓库 [Actions](https://github.com/Protagonistss/Slate-CI/actions) 构建，在对应 Run 的 **Artifacts** 中下载。
